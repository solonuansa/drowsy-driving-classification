# drowsy-driving-classification
This repository contains code and resources for a drowsy driving classification project. The goal is to classify whether a driver is drowsy or not based on various features.
## Dataset
The dataset used in this project is available at [Kaggle](https://www.kaggle.com/datasets/akshaybhalotia/drowsy-driving-classification). It contains various features related to driving behavior and drowsiness.
